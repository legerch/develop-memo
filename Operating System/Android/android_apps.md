# 1. Medias
## 1.1. Sync / Backup

- [Autosync][autosync-gplay] ([official website][autosync-home]) allow to synchronize device folders to a cloud provider (_OneDrive, Google Drive, Dropbox, etc..._)

## 1.2. Audiobooks

- [Voice][voice-gplay] ([home repository][voice-repo])

## 1.3. Social

- add boostforreddit
- revancedapp

# 2. Security

- [AppLock][applock-gplay] allow to add a pin/biometric verification to selected apps (native Android devices doesn't have this feature... maybe in [**Android 17**][applock-native])

# 3. Maps

- [CityMapper][citymapper-gplay]: Useful to find our way via local transport (metro, bus) between a point A and a point B

# 4. Administration
## 4.1. Parcels

- [Parcels][parcel-gplay]: useful to track any parcels status

## 4.2. Money

- [Sesterce][sesterce-gplay]: Used to manage balance of expenses between friends/family
